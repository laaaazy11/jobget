# 🎯 Offer捕手 - AI求职智能匹配助手

> 帮助大学生精准匹配岗位，提升简历初筛命中率

## 🚀 在线体验
https://你的用户名.github.io/offer-catcher/

## 🛠 技术栈
- 前端：HTML5 + CSS3 + JavaScript
- AI引擎：DeepSeek Chat API
- 部署：GitHub Pages

## 📖 使用方法
1. 粘贴简历或点击「填入示例简历」
2. 选择预设岗位或手动输入JD
3. 点击「开始智能匹配」
4. 查看AI分析结果
